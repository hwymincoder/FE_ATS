ATS WEB UI Components

Button Action
Nút bấm với nhiều variants và sizes, hỗ trợ asChild để render thành component khác (dùng Slot từ Radix). Đặt tại src/components/ui/button.jsx.

Demo

button.jsx

6 variants: default, destructive, outline, secondary, ghost, link. 4 sizes: default, sm, lg, icon.

import { Button, buttonVariants } from '@/components/ui/button'; // Basic usage <Button>Click me</Button> // With variant & size <Button variant="destructive" size="sm">Delete</Button> // Render as different element (uses Radix Slot) <Button asChild> <a href="/dashboard">Go to Dashboard</a> </Button>

Dependencies Setup
Các thư viện cần cài đặt để sử dụng UI components. Chạy lệnh bên dưới trong thư mục project.

Core Dependencies

# Install base dependencies npm install class-variance-authority clsx tailwind-merge lucide-react # Install Radix UI primitives npm install @radix-ui/react-slot @radix-ui/react-label @radix-ui/react-dialog @radix-ui/react-dropdown-menu @radix-ui/react-select @radix-ui/react-separator

Utils - lib/utils.js

Hàm cn() (clsx + tailwind-merge) dùng để merge classNames. Đặt tại src/lib/utils.js.

import { clsx } from 'clsx'; import { twMerge } from 'tailwind-merge'; export function cn(...inputs) { return twMerge(clsx(inputs)); }

Index Export - components/ui/index.js

Tạo file index.js export tất cả components để import gọn hơn.

// src/components/ui/index.js export * from './button'; export * from './input'; export * from './textarea'; export * from './label'; export * from './badge'; export * from './card'; export * from './separator'; export * from './dialog'; export * from './dropdown-menu'; export * from './select'; export * from './table';

Sau đó có thể import như:

import { Button, Card, Dialog, Table } from '@/components/ui';
